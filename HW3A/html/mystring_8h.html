Appends the first n characters of src to the string pointed to by dest, plus a null terminator. If n is greater than the length of *src, then it just appends over the entirety of src. If n is equal to the length of src, it simply appends over the entirety of src. If n is less than the length of src, only the first n characters of src (and a null terminator) will be appended to dest.

Parameters
destthe destination to concatenate onto
srcthe source to concatenate from
